About This Collection

This SurVis site accompanies a COMP4126 Research Methods Coursework 3 literature review on face super-resolution and blind face restoration.

The collection currently includes 16 papers and is organized to support the screenshot and discussion required in Section 3 of the report.

Collection Focus

  • Task axis: face super-resolution, blind face restoration, and face hallucination.
  • Method axis: attention, GAN, dictionary learning, facial priors, compact models, and reinforcement learning.
  • Evaluation axis: real-world degradation settings, identity preservation, and face-recognition-oriented evidence.

Keyword Schema

Representative BibTeX pattern:

keywords = {task:blind face restoration, method:gan, guidance:facial prior, setting:real-world}

The tag cloud is driven directly by the keywords field in bib/references.bib, so future edits only need to update the bibliography and rerun python update_data.py --once.

Local Preview

From this folder, run python -m http.server 8765 and open http://127.0.0.1:8765/.

The root index.html redirects to src/index.html, matching the structure used in the finished coursework example.
